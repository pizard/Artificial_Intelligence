# 인공 지능 (Artificial Intelligence)
 1. 소개
 	 - 3학년 1학기 인공지능 텀 과제입니다.
 	 - 프로젝트/보고서/실행 화면으로 구성되어 있습니다.
 	 - 과제 진행 중이 아닌 과제가 끝난 후 제작된 관리용 repository입니다.
 
 2. 프로젝트 개발 환경
	 - Eclipse / Java(Version 미상)

 3. 과제 설명
 	 - Four In A Row
 	 	 - Four In A Row 게임의 인공지능을 간단한 딥러닝 기술을 이용하여 제작
 	 	 - 전체 게임을 제작한 것은 아니고 Heuristic 방식을 이용하여 점수를 부여하는 알고리즘 부분만 작성하였습니다.
